What is it?

Frontier is a co-located social game about managing resources, built on with the Angular JS framework and PubNub API. Frontier is played on a shared screen as well as each player's personal screens. Through a web browser the user can start the game on their computer and the players can join via their phone or tablet. Frontier is set in a science fiction world where each player is a different species with differing needs. The twist is that they all have to coexist on the same planet. Frontier is all about balancing your species personal needs with the needs of a planet. Resources give you points, but use too much of them and the planet will suffer!

The Process

The process of designing a game started with brainstorming sessions where the whole group made use of parallel prototyping in order to come up with as many ideas as possible. We finally decided on a co-located social game. After deciding on the games goals and rules it was time to code.

The game was coded using the PubNub API in order to allow for several devices to communicate with each other in real time. The game works by putting players who want to play with each other into the same game channel, which then allows them to vote on issues within their own game instance. The game works essentially as a chat channel between the players, where the players' votes determine what events happen in the game.
