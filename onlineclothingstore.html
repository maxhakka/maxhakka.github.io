What is it?

Online Clothing Store is an prototype online web store for clothing, built with the Angular JS framework. It has lots of features including log in functionality, shopping cart, categories, a search bar and a whole lot more. The store pulls data from a MySQL database using an API built in PHP.

The item page, which allows you to choose size and color as well as showing how many clothes are available in the database.

The item page also includes suggestions on what else you might also like.

The Process

The process of building the web store started with making wireframes that included all the features in the wireframing tool Balsamiq. Before creating the web store we had to make our own MySQL database in order to keep track of the clothes. We also created an API for the database in order to interface with the database.

One of the most challening parts of building the store was using Angular JS as I had no experience with front-end frameworks before the project. Because of that, this was one of the projects where I learned the most about web development.
