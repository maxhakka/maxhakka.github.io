What is it?

BattleNub is an online multiplayer battleship game that can be played in modern web browsers. The game is built with the PubNub API, which allows for submitting data between devices. To play the game simply send your randomly generated game ID to a friend and start battling against each other!

The Process

The idea for the game came from building chatrooms with the PubNub API. By modifying the code for the chatrooms we could build a fully functioning multiplayer game using the same principles. Two players could play against each other by simply joining the same "chat channel" and sending coordinates to each other in order to hit the opponents ships.
